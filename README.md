## store

美食店铺信息分类站 | based on nodejs

```
一个美食店铺信息分享网站. 

基于nodejs, express + mongodb + es6 + jquery + GoogleMap 

实现了: 

多用户登录、注册、个人信息编辑.

店铺的创建、详情、评价、留言、排名、分类和基于地理位置的查询系统.
```
